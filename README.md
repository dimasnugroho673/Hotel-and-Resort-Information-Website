# Hotel-and-Resort-Information-Website
Ini adalah proyek yang berisi tampilan untuk website Resort dan Hotel. Komponen pada tampilan ini dibuat dengan HTML dan CSS Native, dan belum menggunakan Framework apapun. Namun sudah ada tambahan library Javascript yaitu SweetAlert2 pada bagian login
